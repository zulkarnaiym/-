# 💧 Родник

**Интерактивная карта родников и источников Казахстана**

## Быстрый старт

```bash
# Установить зависимости (один раз)
pip install flask PyJWT

# Запустить сервер
python3 server.py

# Открыть в браузере
# http://localhost:3000
```

## Требования

- Python 3.8+
- Пакеты: `flask`, `PyJWT`

```bash
pip install flask PyJWT
```

## Структура проекта

```
/project
  server.py        ← Flask + все маршруты API
  mines.db         ← SQLite база (создаётся автоматически при первом запуске)
  tile_cache/      ← Кэш тайлов карты (создаётся автоматически)
  public/
    index.html     ← SPA (3 страницы: лендинг, авторизация, карта)
    app.js         ← Логика: карта, API, формы
    style.css      ← Стили (тёмная тема)
    leaflet.js     ← Leaflet.js v1 (локальный файл)
    leaflet.css    ← Leaflet.css (локальный файл)
```

## API

| Метод  | URL                        | Описание                              |
|--------|----------------------------|---------------------------------------|
| POST   | /api/auth/register         | Регистрация                           |
| POST   | /api/auth/login            | Вход                                  |
| GET    | /api/mines                 | Все родники + кол-во заметок          |
| POST   | /api/mines                 | Добавить родник (авторизация)         |
| PUT    | /api/mines/:id             | Редактировать (только владелец)       |
| DELETE | /api/mines/:id             | Удалить (только владелец)             |
| GET    | /api/mines/:id/notes       | Заметки родника                       |
| POST   | /api/mines/:id/notes       | Добавить заметку (авторизация)        |
| GET    | /api/tiles/:z/:x/:y.png    | Прокси тайлов карты (Яндекс → OSM)   |

## Поля родника

| Поле            | Тип     | Описание                                        |
|-----------------|---------|-------------------------------------------------|
| name            | string  | Название родника (обязательное)                 |
| type            | string  | Тип: Родник / Источник / Скважина / Колодец     |
| status          | string  | Статус: Активный / Законсервирован / Закрыт     |
| depth_m         | integer | Глубина в метрах (для скважин/колодцев)         |
| year_opened     | integer | Год открытия / описания                         |
| description     | string  | Описание                                        |
| lat, lng        | float   | Координаты                                      |
| ph              | float   | Водородный индекс (pH)                          |
| mineralization  | float   | Общая минерализация, мг/л                       |
| conductivity    | float   | Удельная проводимость, мкСм/см                  |
| hardness        | float   | Общая жёсткость, мг-экв/л                       |
| temperature_c   | float   | Температура воды на выходе, °C                  |
| sediment        | string  | Характер примесей / осадок                      |
| water_quality   | string  | Статус качества: Подходит / Анализ / Отклонён   |

## Сброс базы данных

Для пересоздания базы с обновлёнными seed-данными:

```bash
# Windows
del mines.db

# Linux/Mac
rm mines.db

# Затем перезапустить сервер — база создастся автоматически
python3 server.py
```

## Описание

Родник — интерактивная карта родников и природных источников Казахстана. Включает:

- 🗺️ Карту Leaflet.js с Яндекс-тайлами через серверный прокси
- 💧 45 реальных родников и источников по всем регионам Казахстана
- 🔬 Показатели качества воды: pH, минерализация, проводимость, жёсткость
- 🔐 JWT-аутентификацию
- 📝 Систему заметок к каждому роднику
- 🎨 Тёмный дизайн
